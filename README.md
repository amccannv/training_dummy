# RS3 Rotation Trainer

Browser-based keybind and rotation training tool for RuneScape 3. Helps build muscle memory for keybinds and practice PvM rotations without logging into the game.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Usage

1. **Edit Keybinds** — assign keyboard shortcuts to each action
2. **Practice** — select a rotation and press the correct keys for each step
3. **Results** — review hits, misses, and accuracy

Built actions, keybinds, and practice history are stored in your browser's localStorage.

## Stack

React 18 · TypeScript · Vite · Zustand

No backend, no database, no accounts.
